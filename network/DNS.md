### DNSとは？
ドメインネームシステムの略であり、ドメイン名から正しいIPアドレスを見つけるために使用される。

### DNS階層について。(https://www.example.com./home.html)それぞれの階層
- **https://** : Scheme
- **www** : Subdomain
- **example**: 2nd-Level Domain
- **com** : Top-Level Domain
- **./** : Root
- **home.html** : Page name

### DNS解決プロセスの順番

1. ドメインをブラウザに入力する
2. ローカルのDNSキャッシュをチェックして、IPアドレスを既に認識しているか確認する
3. 見つからない場合は、再起DNSサーバーに問い合わせる。通常はインターネットサービス・プロバイダまたはサードパーティのDNSサービスによって提供される
4. 再帰DNSサーバーは、ルートサーバーを指す
5. TLDネームサーバーは、権威DNSサーバーに対してクエリを投げる
6. 権威ネームサーバーは、IPアドレスを返す
7. 再帰DNSサーバーはIPアドレスをコンピュータに渡す。
